# MUDO 🤫 (日本語)

[![License: CC0-1.0](https://img.shields.io/badge/License-CC0--1.0-lightgrey.svg)](https://creativecommons.org/publicdomain/zero/1.0/)

シンプルで高速、かつ堅牢な YouTube Music ダウンローダーです。軽量な実行ファイルに同梱されたローカル Web サーバーとして動作し、ブラウザを通じて直感的な操作が可能です。

## 主な機能

- **高品質ダウンロード**: 最大 320kbps の MP3 または可逆圧縮の WAV 形式に対応。
- **スマートなメタデータ**: アーティスト、タイトル、ID3 タグを MP3 ファイルに自動埋め込み。
- **高画質カバーアート**: **iTunes Search API** を使用して、高解像度のジャケット画像を自動取得。
- **自動クリーンアップ**: 変換後に不要な一時ファイルを自動的に削除。
- **効率的なフォルダ整理**: プレイリスト用のフォルダを自動作成し、ファイル名を `アーティスト - タイトル` 形式で保存。
- **ライブラリ管理**: アプリ内のインターフェースからダウンロード済みフォルダを直接管理・展開。

## 使い方

1. [最新リリース](https://github.com/MidoriAKA/simple-music-downloader/releases/latest)から `MUDO.exe` をダウンロードして実行します。
2. 既定のブラウザで操作画面が自動的に開きます。
3. **YouTube または YouTube Music のプレイリスト URL** を貼り付けます。
4. フォーマット（MP3/WAV）や保存先サブフォルダ（任意）を選択します。
5. **Start Download** をクリックして開始します。

## 動作環境

- Windows 10/11 (x64)
- 外部依存関係なし（FFmpeg および yt-dlp は同梱されています）。
